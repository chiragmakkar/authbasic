<---------BASIC AUTH API for SOCIAL COPS--------->

To install server - npm install
To start server - npm start 
To test api(s) - npm test 

Routes at port 3000 :

1. Authentication -
	address : /login
	method : POST
	input : username, password

2. JSON Patch -
	address :/patch 
	method : POST
	input : object, patch(s) separated by , (both JSON)
3. Thumbnail -
	address : /thumbnail
	method : POST
	input : uri
